## [0.1.0] - 2024-01-01

### Added

#### Backend
- Finance module with chart of accounts and journal entries
- Inventory module with products, warehouses, and stock movements
- Sales module with customers and sales orders
- Purchasing module with vendors and purchase orders
- Manufacturing module with BOMs and work orders
- HR module with employees, attendance, and payroll
- JWT-based authentication with role-based access control
- SQLite database with migrations

#### Frontend
- React admin dashboard with Tailwind CSS
- Login and registration pages
- Dashboard with statistics
- Finance page for accounts and journal entries
- Inventory page for products and warehouses
- Sales page for customers and orders
- Purchasing page for vendors and POs
- Manufacturing page for BOMs and work orders
- HR page for employees and attendance

#### Documentation
- README with quick start guide
- API documentation
- Deployment guide
- Contributing guide
- AGENTS.md for AI assistants
